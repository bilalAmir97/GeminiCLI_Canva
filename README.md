# GeminiCLI Canva Hero Section

This project is a recreation of the Canva hero section, built with pure HTML5 and CSS3 through GEMINI CLI


## Features

- **Responsive Design:** The hero section is fully responsive and adapts to different screen sizes.
- **Modern Animations:** The hero section includes modern animations on the heading, subheading, buttons, and background icons.
- **Pure HTML/CSS:** The project is built with only HTML5 and CSS3, with no JavaScript or external frameworks.
- **Pixel-Perfect Recreation:** The hero section is a close recreation of the Canva hero section, with attention to detail in fonts, colors, and layout.

## Technologies Used

- HTML5
- CSS3
- Git
- GitHub
- Gemini CLI
